# Up to Wine — Datos de vinos

Fichas técnicas de los vinos de [uptowine.cl](https://uptowine.cl) para la app Up to Wine.

- `wines.json` — catálogo con añadas, notas de cata y recomendaciones del sommelier.
- `scrape.js` — genera `wines.json` leyendo las fichas públicas de la tienda.
- Se actualiza automáticamente cada 6 horas vía GitHub Actions.
